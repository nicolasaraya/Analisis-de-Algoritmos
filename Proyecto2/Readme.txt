Para ejecutar el programa, primero se debe compilar utilizando: g++ +.cpp

Luego se debe ejecutar siguiendo la estructura ./a.out a b c d e f
	a: Numero de filas del grafo de grilla
	b: Numero de columnas del grafo de grilla
	c: Numero minimo de pesos de las aristas
	d: Numero maximo de pesos de las aristas
	e: Numero de filas del laberinto
	f: Numero de columnas del laberinto

Una vez ejecutado, el programa imprimira el Grafo de grilla generado, el MSF y el laberinto.
